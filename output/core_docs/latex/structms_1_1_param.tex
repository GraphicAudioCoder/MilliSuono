\doxysubsection{Detailed Description}
Represents a named parameter of a \doxylink{classms_1_1_node}{Node}. 

A parameter stores a name and a corresponding \doxylink{namespacems_a151de2fccda8383fc510229a1fcfc8e2}{Control\+Value}. Parameters can represent any configurable property such as gain, frequency, or mode.
